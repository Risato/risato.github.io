--- layout: archive title: "Projects" permalink: /projects/ author_profile: true --- I dabble with various data service projects for professional and student organizations.


Interactive Leaflet Web Map

Interactive web mapping visualization of woody biomass facilities. Includes a wide array of information regarding wood biomass utilization and capacity in the State of California. Archival leaflet and javascript web map containing layers for a more intuitive user experience.

Searchable Tables

Javascript Tables developed for UC Berkeley Forestry Camp to create a searchable list of past students.

Interactive CartoDB Web Map

Dynamic Cartodb Map with javascript responsive interface. Exercise tested for a class to implement a responsive map withe Cartodb's hosting services. Updated annually.

Panorama Comparison

Immersive 360° panoramic photo comparisons. Taken before and after fuels treatment at various locations in Sierra Mixed Conifer and Chaparral-Chemise ecosystems.

Immersive Forest Tour

Panoramic tour of UC's Baker Forest property in Plumas County. Built using PTGUI and Photoshop. A pilot project developed in 2009 and no longer maintained.

Hobby Projects

Collection of woodworking and Raspberry Pi projects. Page under development.
