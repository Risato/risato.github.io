I dabble with several public facing Data Service projects for professional and student organizations. I would be happy to discuss them in more detail.

  • Interactive Leaflet Web Map

    Leaflet-based interactive javascript web map containing layers for a more intuitive user experience. Includes a wide array of information regarding wood biomass utilization and capacity in the State of California.

  • Searchable Tables

    Javascript Tables developed for UC Berkeley Forestry Camp to create a searchable list of past students.

  • Interactive CartoDB Web Map

    Dynamic Cartodb Map with javascript responsive interface. Exercise tested for a class to implement a responsive map withe Cartodb's hosting services. May not be updated.

  • Panorama Comparison

    Immersive 360° panoramic photo comparisons. Taken before and after fuels treatment at various locations in Sierra Mixed Conifer and Chaparral-Chemise ecosystems.

  • UC Berkeley Forestry Club

    UC Forestry Club is the student chapter of the Society of American Foresters. It is one of the oldest clubs on campus, pre-dating the discipline at UC Berkeley. I helped with content and design updates to their website, integrating many previously archived resources and adding new fucntionality such as an integrated paypal payment system.
